## üöÄ Selected Work

### [PayChain](https://github.com/DeepakPratapa/paychain) - Blockchain Escrow with Microservices
**5 FastAPI services. Dual refund logic smart contracts. Zero trust required.**

Engineered a production-ready freelance escrow platform with passwordless MetaMask authentication, JWT + Redis token blacklisting, WebSocket-triggered React Query invalidation, and Solidity contracts with automated milestone-based fund release. The architecture demonstrates advanced distributed systems design with centralized security guards spanning all microservices.

**Stack:** `React` `FastAPI` `Solidity` `PostgreSQL` `Redis` `Docker` `WebSockets`

---

### [Blockchain-Verified Portfolio](https://github.com/DeepakPratapa/personal-portfolio-blockchain)
**Every file hash stored on Polygon. Public auditability. Tamper-proof deployment.**

Built a revolutionary two-layer verification system where all source code, dependencies, and build artifacts are cryptographically hashed and stored on-chain. Automated CI/CD pipeline includes pre-deployment security scanning, reproducible build verification, and post-deployment integrity validation. Anyone can verify the deployed code matches its on-chain hash.

**Stack:** `Solidity` `Hardhat` `React` `Polygon` `SHA-256` `CI/CD`

---

### [Serverless Image Processor](https://github.com/DeepakPratapa/serverless-img-proc-aws)
**Event-driven AWS infrastructure. Zero server management. Auto-scaling.**

Designed scalable image processing platform using Lambda, S3, API Gateway, and Amplify. CloudFormation IaC ensures reproducible deployments. GitHub Actions automate testing and deployment. CloudWatch monitoring provides real-time performance insights. IAM roles and S3 policies enforce least-privilege security.

**Stack:** `AWS Lambda` `S3` `API Gateway` `CloudFormation` `GitHub Actions`

---

### [Firebase Task Manager](https://github.com/DeepakPratapa/serverless-task-manager-firebase)
**Real-time collaboration. Automated notifications. Serverless workflows.**

Full-stack task management platform with Firestore real-time sync, Cloud Functions for email automation, and Firebase Auth for security. Event categorization and filtering enable efficient task organization. Demonstrates serverless architecture patterns and real-time data synchronization.

**Stack:** `React` `Firestore` `Cloud Functions` `Firebase Auth`
